Chinese Dictator - Visit site - Github

Made with: Ruby, Rails, Heroku, HTML5/CSS3/jQuery, HAML/SASS

A tool to help overcome a specific challenge for learners of Mandarin Chinese: listening skills. The syllables and tones don't match what we're used to, and it takes practice to be able to differentiate them. It even allows you to filter data by troubling syllable initials/finals, such as 'q' and 'ch'. Data is scraped from ChinesePod.
